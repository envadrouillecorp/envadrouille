1. Installation

  • Upload all envadrouille files on your server, e.g., in a directory named /envadrouille/
  • Open your web browser and go to the administration ( /envadrouille/admin ).
  • Validate the installation. You can leave all fields to their default value. The only important point is to set a global administration settings > password. If you leave the password field empty, anyone can modify your galleries. We also recommend that you check the "Check update" checkbox to be notified when a new version of the gallery is available.
  • You're done, now you can add galleries (see next section).

2. Add a gallery

  • Upload photos in your 'picture directory'. E.g., create a directory '/envadrouille/pics/2013-01-24 My Gallery/' and put pictures in it.
  • Go to the index of the administration (if you are already in the administration, click on the green button on the top left of the administration).
  • Complete the description of one gallery and click the update button.
  • Done!
  • If you want to update a gallery, click on the directory name and modify what you want.
  • You can also upload .gpx files (or simply put them with photos in your pictures directory) to show the place you visited, hide the directory (it will not appear in the index of the gallery), or 'star' it (will add a little star on the gallery in the index so that the gallery is more visible).
  • Don't forget to update a gallery after adding pictures in it.

3. Supported file types

You can put files with these extensions in your galleries. Other files will be ignored.
  • Pictures: jpg, png, gif
  • Videos: mp4, ogg, ogv
  • GPS files: gpx

4. Face recognition

The gallery can recognize faces in your photos. To start face recognition:
  • Activate face recognition in the administration
  • Register to one of the 'face recognition' service (rekognition.com, api.animetrics.com, ...)
  • Go to the face recognition tab (blue button next to the option button at the top of the screen - should appear once face recognition is activated).
  • Click the help button and follow the instructions.
  • Note that only pictures already added to the gallery are eligible for face recognition.
  • Keep in mind that:
    • Your photos will be send to the selected face recognition provider. Read their policy concerning the privacy of recognized faces.
    • Face detection can take some time, be patient.

1. Installation

  • Envoyez tous les fichiers de la galerie envadrouille sur votre serveur, par exemple dans un dossier /envadrouille/
  • Ouvrez votre navigateur et allez dans l'administration ( /envadrouille/admin ).
  • Validez l'installation.
    • Le seul point important est de remplir le champ global administration settings > password (ou "mot de passe" dans la section "Paramètres de l'administration" en français). Lorsque le mot de passe est vide, tout le monde peut modifier la galerie.
    • Pour passer l'administration en français, utilisez le paramètre language dans les paramètres globaux de l'administration.
    • Vous pouvez également passer la galerie elle même en français (en plus de l'administration). Dans les paramètres de la galerie.
    • Vous pouvez cocher la cache "Check update" ("Rechercher les mises à jour") pour être notifié lorsque des mises à jour pour la galerie sont disponibles (recommandé).
  • Vous pouvez maintenant ajouter votre première galerie (cf. section suivante) !

2. Ajouter une galerie

  • Uploadez des dossiers photos dans le "répertoire photos". Par exemple, créez un dossier '/envadrouille/pics/2013-01-24 My Gallery/' et mettez des photos dedans.
  • Allez dans l'index de l'administration (bouton vert en haut à gauche).
  • Complétez la description d'un dossier et mettez le à jour (bonton "mise à jour" ou "update")
  • Voilà !
  • Vous pouvez modifier une galerie déjà ajoutée en cliquant sur son nom.
  • Vous pouvez ajouter des fichiers .gpx (uploadez le avec vos images dans le même dossier), des vidéos, etc. Vous pouvez cacher une galerie, ou lui rajouter une étoile (qui permet de la rendre plus visible sur la page principale de la galerie).
  • N'oubliez pas de mettre à jour une galerie lorsque vous rajoutez des photos.

3. Fichiers reconnus

La galerie reconnait les extensions suivantes :
  • Photos : jpg, png, gif
  • Vidéos : mp4, ogg, ogv
  • GPS : gpx

4. Reconnaissance faciale

La galerie reconnait les visages de vos photos. Pour activer la reconnaissance faciale :
  • Cochez la cache correspondance dans les options
  • Inscrivez vous sur un site de reconnaissance faciale (rekognition.com, api.animetrics.com, ...)
  • Allez dans l'onglet de reconnaissance faciale.
  • Cliquez sur le bouton d'aide et suivez les instructions.
  • Attention: seules les images déjà ajoutées dans une galerie seront reconnues.
  • Notez également :
    • Vos photos seront envoyées au service de reconnaissance faciale sélectionné. Vérifiez leur politique concernant la vie privée.
    • Certains services de reconnaissance faciale sont très lents. Soyez patients.
